# Halix Two-Pass Assembler (C++)

A simple C++ implementation of a **two-pass assembler** 

---

make sure to include the assembler_block_v2.cpp and assembler_block_v2_tests.cpp on the same directory that you are running the commands

## 🔧Step-1: Compile Assembler

g++ -std=c++17 -O2 assembler_block_v2.cpp -o assembler


###  Step-2: Unit Testing

g++ -std=c++17 -O2 assembler_block_v2_tests.cpp -o assembler_tests

###  Step-3: Run Tests

./assembler_tests

###  Step-4:Run Assembler

./assembler any_inputfile.hal

###  Step-5:Run Dr.Jones's emulator (halix_v25.crun) to see if .hlt file will be generated. Make sure you are testing the .hlx file generated by the assembler this time.

./halix_v25.crun anyfile.hlx
